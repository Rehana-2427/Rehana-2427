## 🏆 Live Projects

### 🔷 [JobBox.one](https://jobbox.one) — AI-Powered Job Portal
> Full stack job portal with AI-driven job matching and recommendations

- **Backend:** Spring Boot · Microservices · Spring Security · JWT · Redis · MySQL · MongoDB
- **Frontend:** React.js
- **Cloud:** AWS EC2 · S3 · RDS · IAM
- **Features:** AI job recommendations · Resume upload to S3 · Role-based access · Microservices architecture

---

### 🔷 [SignBook.co](https://signbook.co) — Digital Signature Platform
> End-to-end digital signature and document management platform

- **Backend:** Spring Boot · Microservices · Redis Caching · MySQL · MongoDB
- **Frontend:** React.js
- **Integrations:** Calendly scheduling integration · Digital signature workflow
- **Features:** Document upload · Signature tracking · Redis caching for performance · REST APIs

---

### 🔷 AWS Vault — Cloud File Storage App
> Spring Boot application demonstrating AWS services integration

- **Stack:** Spring Boot · AWS EC2 · S3 · RDS MySQL · IAM Policies
- **Features:** User file/resume upload to S3 · Structured data in RDS · IAM role-based team access · VPC with public/private subnet separation
- **Learning:** EC2 deployment · S3 SDK integration · IAM least privilege · VPC network architecture

---

### 🔷 [GoBig.cc](https://gobig.cc) — Open Design Network
> Design collaboration platform built with modern Next.js stack

- **Stack:** Next.js 14 (App Router) · TypeScript · Prisma ORM · Supabase PostgreSQL · NextAuth v5
- **UI:** TanStack Query · shadcn/ui · Tailwind CSS
- **Deployment:** Vercel · Supabase database · Agentic AI assisted development

### 🔷 [Portfolio](https://portfolio-seven-taupe-33.vercel.app/) — Personal Portfolio Website
